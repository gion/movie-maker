CSSPlugin — class, package com.greensock.plugins
With the help of the CSSPlugin, TweenLite and TweenMax can animate almost any css-related property of DOM elements including the obvious things like width, height, margins, padding, top, left, and more plus more interesting things like transforms (rotation, scaleX, scaleY, skewX, skewY, x, and y), colors, percentages, opacity, and more.
CSSRulePlugin — class, package com.greensock.plugins
